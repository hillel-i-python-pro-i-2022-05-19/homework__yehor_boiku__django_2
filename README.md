## Docker

Use services in dockers.

### Run

Just run

```shell
make d-run
```

### Run extended

Shutdown previous, run in detached mode, follow logs

```shell
make d-run-i-extended
```

### Stop

Stop services

```shell
make d-stop
```

### Purge

Purge all data related with services

```shell
make d-purge
```
